```

}
```

3) Для списка словарей с данными о товара
```
products = [
    {"id": 1, "price": 100, "category": "A"},
    {"id": 2, "price": 200, "category": "B"},
    {"id": 3, "price": 50, "category": "A"}
]
```
Рассчитать:
- Среднюю цену по категориям

- Общую стоимость всех товаров
