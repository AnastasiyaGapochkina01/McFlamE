```
1. Five Finger Dead Punch
2. Apocalyptica
```
4) **Дописать** в файл data.txt строку, которую введет пользователь (input)
5) Написать функцию, которая подсчитает, сколько строк содержит файл data.txt
6) Задан список
```python
dragons = [
    "venomous dragon",
    "infected dragon",
    "forest dragon",
    "energy dragon"
]
```
Записать элементы списка в файл dragons.txt построчно

7) Считать построчно файл dragons.txt и создать на его основе словарь dragons_dict, где ключом будет номер по порядку, а значением - вид дракона

```
venomous dragon — дракон, чьи зубы и когти пропитаны смертельным ядом. Он предпочитает резкие атаки и оставляет после себя облако ядовитого пара.
infected dragon — иссохшее и заражённое чародейской болезнью существо, способное распространять порчу вокруг себя. Его чешуя покрыта трещинами, из которых просачивается тьма.
forest dragon — хранитель дикой природы, с кожей цвета мха и крыльями, напоминающими листву. Этот дракон защищает лес и черпает силу из деревьев.
energy dragon — воплощение чистой силы, излучающей свет и электрические разряды. Он питается энергией гроз и способен управлять молниями.
```
Считать файл dragons-description.txt построчно и на его основе создать словарь dragons_descript, где ключом будет тип дракона, а значением - описание

9) Написать функцию, которая запросит у пользователя номер дракона и выведет описание этого типа дракона

#### Подсказки к 8 и 9 задачам
<details>
  <summary>Разделение строки</summary>
  
  ```python
  dragons_descript = {}
  line = "venomous dragon — дракон, чьи зубы и когти пропитаны смертельным ядом. Он предпочитает резкие атаки и оставляет после себя облако ядовитого пара."
  parts = line.split("—", 1) 
  if len(parts) == 2:
    name = parts[0].strip() 
    description = parts[1].strip()  
    dragons_descript[name] = description

  for key, value in dragons_descript.items():
    print(f"Тип - {key}\nОписание: {value}")
  ```
  #### пояснение
  `parts = line.split("—", 1)` разбивает строку на две подстроки, используя как разделитель символ -\
  результатом является _**список**_
  
  если строка успешно разбилась на 2 (`if len(parts) == 2`), то можем доставать имя по индексу 0 (`parts[0]`), удаляя пробелы _в начале и в конце_ (`.strip()`)

  аналогично с description
